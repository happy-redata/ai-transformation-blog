Let's start with what most people know. <Voice text="Chat G-P-T">ChatGPT</Voice>, <Voice text="Gemini">Gemini</Voice>, <Voice text="Claude">Claude</Voice> — the large language models you access via a browser.

But they have limitations:

| Challenge | Consequence |
|-----------|-------------|
| **Copy-paste workflow** | You lose context every time you switch windows |

It's like having a brilliant advisor who sits in another room and can never see what you're actually working on.

## GitHub Copilot: Your First <Voice text="A-I">AI</Voice> Assistant in the Editor

<Voice text="GitHub Co-pilot">GitHub Copilot</Voice> is an <Voice text="A-I">AI</Voice> assistant that lives *inside* your code editor. It can see your files, understand your context, and suggest code in real-time.

It's like having an experienced colleague looking over your shoulder saying: *"Try writing it like this..."*

### How it works in practice

Imagine you're writing a function. You start by typing:

```javascript
function calculateTax(
```

Suddenly you see gray text appear — a suggestion from <Voice text="Co-pilot">Copilot</Voice>:

```javascript
function calculateTax(income, taxRate) {
  return income * taxRate;
}
```

Press **Tab**, and the suggestion is inserted. Press **Escape**, and you ignore it.

It's **autocomplete on steroids**.

## Ask Mode vs. Agent Mode: Understanding the Difference

<Voice text="GitHub Co-pilot">GitHub Copilot</Voice> has two fundamentally different ways of working. And understanding the difference is key to using <Voice text="A-I">AI</Voice> effectively.

### Ask Mode: You ask, <Voice text="A-I">AI</Voice> answers

In **Ask Mode**, <Voice text="Co-pilot">Copilot</Voice> works like an advanced chat — but *inside your editor*. You can:

- Ask for explanations
- Request code suggestions
- Get help with debugging

But the <Voice text="A-I">AI</Voice> doesn't *do* anything. It only suggests. You still have to copy, paste, edit yourself.

### Agent Mode: <Voice text="A-I">AI</Voice> Takes Action

**Agent Mode** is something completely different. Here the <Voice text="A-I">AI</Voice> can:

- **Edit files** directly in your project
- **Create new files** where needed
- **Run commands** in the terminal
- **Install packages** and dependencies
- **Debug** by reading error messages and reacting

<View>
Here's the crucial difference:
</View>

| Function | Ask Mode | Agent Mode |
|----------|----------|------------|
| Gives suggestions | Yes | Yes |
| Edits files | No | **Yes** |
| Runs commands | No | **Yes** |
| Creates files | No | **Yes** |
| Requires approval | No | Yes (you confirm changes) |

In Agent Mode, you're no longer alone at the wheel. You have a co-pilot who can actually *fly*.
